# 🌌 Novadeed | Galactic Registry & Exploration
StarBound is a next-generation celestial navigation and star registration platform. It combines real-time astronomical data with a high-premium, aerospace-inspired interface to bring the universe to your fingertips.
### ✨ Key Features
- **3D Celestial Map**: Immersive space exploration powered by Three.js and React Three Fiber.
- **NASA Integration**: Real-time exoplanet and stellar data pulled directly from NASA's archives.
- **Digital Registry**: Secure your legacy by claiming and naming stars in the galactic archive.
- **AstroX Design System**: A sharp, technical UI/UX designed for a premium command-center feel.
- **Full i18n**: Native support for English and Turkish across all layers of the experience.
### 💻 Tech Stack
- **Frontend**: Vite, React, Three.js, GSAP (Animations), Zustand (State)
- **Backend**: .NET 8 Web API, Entity Framework Core, SQLite
- **Security**: JWT Authentication, Google OAuth Integration
